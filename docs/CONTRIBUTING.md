# Contributing to GravelMapper

Thanks for wanting to help! GravelMapper is built by riders for riders.

---

## How to contribute

### Reporting a bug
1. Open an Issue on GitHub
2. Describe what you expected vs what happened
3. Include your device type and OS version

### Suggesting a feature
1. Open an Issue with the label `enhancement`
2. Describe the problem it solves
3. Bonus: include a rough sketch or example

### Submitting code

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Commit: `git commit -m "Add: short description of change"`
5. Push: `git push origin feature/your-feature-name`
6. Open a Pull Request

---

## Commit message style

| Prefix | Use for |
|---|---|
| `Add:` | New feature or file |
| `Fix:` | Bug fix |
| `Update:` | Change to existing feature |
| `Remove:` | Deleting something |
| `Docs:` | README or documentation only |

---

## Questions?

Open an Issue and tag it `question`. Happy to help.
